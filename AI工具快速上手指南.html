👨‍💻 程序员：用Cursor写代码

工具：Cursor（$20/月，最强编程工具）

第一步：下载安装

  1. 访问：cursor.so
  2. 点击 Download
  3. 选择你的系统（Windows/Mac/Linux）
  4. 下载完成后安装
  5. 打开Cursor

第二步：基础设置

  1. 首次打开，选择"导入VS Code配置"（如果你用过VS Code）
  2. 登录账号（用GitHub或Google登录）
  3. 点击右上角设置图标
  4. 选择AI模型：推荐选"Claude 4"

第三步：创建.cursorrules文件

  1. 在项目根目录创建文件：.cursorrules
  2. 复制粘贴这个模板：
# 我的代码规则
- 所有代码必须有注释
- 使用清晰的变量名
- 函数要有完整的错误处理
- 每个函数只做一件事

第四步：开始写代码

方法1：用Ctrl+L对话

  1. 按 Ctrl+L（Mac是Cmd+L）
  2. 输入："创建一个Python函数，读取Excel文件并返回数据"
  3. AI会生成完整代码
  4. 点击"插入代码"按钮

方法2：用Ctrl+K内联编辑

  1. 写一段代码
  2. 选中代码
  3. 按 Ctrl+K（Mac是Cmd+K）
  4. 输入："添加错误处理"
  5. AI自动修改代码

方法3：用@符号引用文件

按Ctrl+L，然后输入：
"@app.py 这个文件的登录逻辑有什么问题？"

AI会读取app.py文件并分析
实际例子：创建用户注册接口
  1. 按Ctrl+L
  2. 输入："创建一个Express API接口 POST /register，接收email和password，验证邮箱格式，密码加密用bcrypt，保存到MongoDB，返回JWT token，添加完整错误处理"
  3. AI生成完整代码
  4. 点击"插入代码"
  5. 完成！
✅ 你已经学会用AI写代码了！

💡 程序员进阶：理解4个核心概念

📌 提示：以下概念是为了让你更好地使用AI编程工具，不理解也不影响基本使用！

1. MCP (Model Context Protocol) = 让AI能读你的文件

简单理解：就像给AI装了"USB接口"，AI可以直接读取你电脑的文件，不用复制粘贴

实际作用：

  • Cursor通过MCP读取整个项目文件夹
  • Claude Desktop通过MCP连接数据库
  • 不用手动复制代码，AI自动理解项目结构

2. Agent = AI自动干活

简单理解：你说"修复所有bug"，AI自动完成：找bug → 修复 → 测试 → 生成报告

实际作用：

  • Cursor的Background Agent可以自动重构整个项目
  • bolt.new可以自动生成完整Web应用
  • 不是简单回答问题，而是真正完成任务

3. Prompt = 怎么跟AI说话

简单理解：就是你给AI的指令，说得越清楚，AI做得越好

对比：

❌ 差的Prompt："帮我写代码"
→ AI不知道写什么，随便给个例子

✅ 好的Prompt："用Python写一个读取Excel文件的函数，要求：
• 输入文件路径
• 返回DataFrame
• 处理文件不存在的错误
• 添加函数注释"
→ AI生成精确符合需求的代码

4. Rules (.cursorrules) = 让AI遵守规矩

简单理解：创建 .cursorrules 文件，写上你的编码规范，AI自动遵守

实际作用：

  • 统一团队代码风格（4空格 or 2空格？）
  • 强制类型注解（TypeScript strict模式）
  • 自动添加错误处理
  • 禁止不安全的代码（eval、SQL注入）
示例 .cursorrules 文件：
# 我的代码规则
- 所有代码必须有注释
- 使用TypeScript strict模式
- 函数要有完整的错误处理
- 变量名要有意义，不用a、b、c
💡 记住：这些概念是让你用得更好，但不理解也能正常使用工具。慢慢实践中就能理解了！
